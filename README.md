Rougue
======

CS196 Rougue Project

Controls:  
WASD or Arrow Keys Move NSEW or diagonally.  
F           - Fireball 
C           - Clayball/Mudball 
X or P      - PoisonCloud
H           - Heal
Left Click  - Magic Bullet towards click location
Spacebar    - Pick up item
< and >     - go up and down floors, respectively
